# Password Strength Analyzer

This project is a Python-based password strength analyzer that evaluates user input based on common security criteria.

## Features
- Checks password length (minimum 8 characters)
- Detects uppercase and lowercase letters
- Detects numbers and special characters
- Provides feedback to improve password strength
- Assigns a strength score based on security rules

## Technologies Used
- Python

## Purpose
This project was built to practice programming logic and explore basic cybersecurity concepts related to password strength.

## How to Run
1. Download the file
2. Run using Python:
   python password_strength_analyzer.py
